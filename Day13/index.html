Position Properties in CSS.

CSS Position Properties Demonstration

Static

Default positioning. Element are placed in normal document flow



Relative

Moving relative to its normal position



Absolute

Position Absolute to the nearest positioned ancestor(container).



fixed

Stay fixed relative to the viewport even when scrolling.



Sticky

Scroll inside this contaier to see this sticky box. it behaves relative utill the top of the container reaches the viewport, then it "sticks.
